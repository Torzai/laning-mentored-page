## Cómo deployar en Vercel (1 minuto)

### Opción 1: GitHub + Vercel (automático)

```bash
git init
git add .
git commit -m "Landing mentorship"
git branch -M main
git remote add origin https://github.com/TUUSUARIO/mentoria-landing.git
git push -u origin main
```

Luego:
1. Abre https://vercel.com
2. Click "Import Project"
3. Selecciona tu repo
4. Click "Deploy"
5. Listo en 30 segundos

### Opción 2: Vercel CLI

```bash
npm i -g vercel
vercel
# Sigue las instrucciones
```

### Opción 3: Drag & Drop

1. Ve a https://vercel.com/new
2. Drag & drop esta carpeta
3. Deploy
